## First Time Setup

- Make sure you have [Node 22.x](https://nodejs.org) (Ideally using [nvm](https://github.com/nvm-sh/nvm))
- Install dependencies `npm i`

## Running the project

```bash
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```
