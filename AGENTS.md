# AGENTS

Minimal guidance for automation and AI agents working in this repo.

## Project

- Purpose: SvelteKit site to help people find shelter fast during weather emergencies.
- Stack: SvelteKit (static adapter) + Vite, TypeScript, Tailwind CSS v4, Vitest, Playwright, Storybook, shadcn-svelte.

## Setup

- Node: 24.11.0 (see `.nvmrc`)
- Install: `npm i`

## Common Commands

- Build: `npm run build`
- Tests
  - `npm run test:unit`: unit tests
  - `npm run test:e2e`: Playwright e2e
  - `npm run test`: runs both unit and e2e tests
- Lint: `npm run lint`
